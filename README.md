# 《でびるコネクショん》非官方中文化補丁

本專案是由粉絲共同維護推出的《でびるコネクショん》（Devil Connection）繁體/簡體中文化補丁。
目前文本已翻譯完畢，但可能仍有少數錯漏（如翻譯錯誤、遺漏、語句不通順等），歡迎到 [Steam 討論區](https://steamcommunity.com/app/3054820/discussions/0/671726388306530312/)提交留言反饋。

## ✨ 補丁特點

* **保留原作韻味**：官方在角色命名上非常有特色，故特意保留遊戲角色名稱不進行翻譯。
* **專注劇情文本**：僅翻譯遊戲對話與劇情文本，不包含 UI 介面、圖像等內容。
* **在地化潤色**：採用在地慣用語進行潤色，非生硬的機器翻譯。

## ⚙️ 使用方法

### ▶️ 安裝步驟

**簡體中文用戶**：
請下載檔案後，參考壓縮檔內的 `安装前请看.txt` 進行安裝。

**繁體中文用戶**：

1. 開啟 Steam，右鍵點擊遊戲 -> 管理 -> **瀏覽本機檔案**，進入遊戲根目錄。
2. 將下載的壓縮檔解壓縮到遊戲根目錄中。
   **📂 檔案結構示意：SteamLibrary\steamapps\common\でびるコネクショん

│

├── Loader.js <-- 載入器核心

├── patch.hdiff <-- 核心差異補丁

├── Mod Loader Installer.bat <-- 安裝腳本

├── hpatchz.exe <-- 補丁應用工具

└── DevilConnection.exe <-- (原本的遊戲執行檔)**
3. 點擊執行 `Mod Loader Installer.bat`，等待安裝視窗跑完。
4. 安裝完成，開始遊玩！ 🎉
   (⚠️若發生無法預期的錯誤，請手動將 `resources`資料夾中的 `app.asar`和 `app.asar.bak`刪除，然後從 steam 執行檢查遊戲完整性後，再次執行中文化安裝。)

### 🔄 遊戲本體更新時（Steam 更新了遊戲）

當 Steam 更新了遊戲檔案，中文化可能會失效，請依照以下步驟操作：

1. 進入 `resources` 資料夾，刪除 `app.asar`。
2. 找到備份檔 `app.asar.bak`，將其重新命名回 `app.asar`。
3. 從 Steam 驗證遊戲完整性或執行更新。
4. 更新完畢後，重新依照上方的「**安裝步驟**」放入中文化檔案。
    * *（若中文化直接被 Steam 更新覆蓋，則刪除 `app.asar.bak` 後重新執行安裝步驟即可）*

### 🆙 中文化補丁更新時

若只是中文化補丁推出新版本（修正翻譯等）：

1. 下載最新版本的補丁壓縮檔。
2. 解壓縮並直接覆蓋遊戲根目錄下的 `plugins` 資料夾即可。
