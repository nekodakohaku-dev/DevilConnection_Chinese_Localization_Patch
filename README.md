# 《でびるコネクショん》非官方中文化補丁

本專案是由粉絲共同維護推出的《でびるコネクショん》（Devil Connection）繁體/簡體中文化補丁。
目前文本已翻譯完畢，但可能仍有少數錯漏（如翻譯錯誤、遺漏、語句不通順等），歡迎到 [Steam 討論區](https://steamcommunity.com/app/3054820/discussions/0/671726388306530312/)提交留言反饋。

## ✨ 補丁特點

* **保留原作韻味**：官方在角色命名上非常有特色，故特意保留遊戲角色名稱不進行翻譯。
* **專注劇情文本**：僅翻譯遊戲對話與劇情文本，不包含 UI 介面、圖像等內容。
* **在地化潤色**：採用在地慣用語進行潤色，非生硬的機器翻譯。

## ⚙️ 使用方法

### ▶️ 安裝步驟

**簡體中文用戶**：
請下載檔案後，參考壓縮檔內的 `安装前请看.txt` 進行安裝。

**繁體中文用戶**：

1. 確認並下載最新版中文化。
2. 將下載的檔案解壓縮到任意磁碟或遊戲根目錄中。
3. 點擊執行 `Mod Loader Installer.bat`，並輸入 1 進行模組的安裝。
4. 安裝完成，開始遊玩！ 🎉

## ❓ 常見問題

### Q1：遊戲本體有更新，我該如何更新？中文化是否會失效？

更新遊戲檔案，中文化會被覆蓋而失效，請依照以下步驟進行更新：

1. 回到本 Github 確認並下載最新版中文化。
2. 進入遊戲根目錄下的 `resources` 資料夾中，刪除 `app.asar`。
3. 找到先前中文化時的備份檔 `app.asar.bak`，將其重新命名回 `app.asar`。
4. 從 Steam 驗證遊戲完整性或直接執行遊戲更新。
5. 更新完畢後，重新依照上方的「**安裝步驟**」進行安裝。

### Q2：Steam 更新了並覆蓋掉遊戲本體，中文化失效了該怎麼辦？

當 Steam 更新了遊戲檔案，中文化會被覆蓋而失效，請依照以下步驟修復：

1. 回到本 Github 確認並下載最新版中文化。
2. 進入遊戲目錄下的 `resources` 資料夾，刪除 `app.asar.bak`。
3. 更新完畢後，重新依照上方的「**安裝步驟**」放入中文化檔案。

### Q3：中文化文本推出了新版本，如何更新？

若只是中文化文本更新（修正翻譯等），不需要重新執行安裝腳本：

1. 下載最新版本的`zh-tw.asar`文件。
2. 替換遊戲根目錄下的`plugins`資料夾內舊的`zh-tw.asar`檔案即可。

    * *（⚠️請務必將 `plugins` 資料夾中，舊版本`.asar`的中文化檔案移除）*

### Q4：裝完中文化遊戲打不開該怎麼辦？

1. 請手動將遊戲跟目錄下`resources`資料夾中的`app.asar`和`app.asar.bak`進行刪除。
2. 從 Steam 驗證遊戲完整性或執行更新。
3. 更新完畢後，重新依照上方的「**安裝步驟**」放入中文化檔案。
4. 若還是不行請到 [Steam 討論區](https://steamcommunity.com/app/3054820/discussions/0/671726388306530312/)提交留言反饋

### Q5：我是否可以改用其他家的漢化？

本中文化補丁支持使用別家的漢化，甚至是同時使用多家漢化：

1. 詳見作者的另一個 [Github 開源項目](https://github.com/nekodakohaku-dev/DevilConnection_Mod_Loader)
